# CURRENT PROGRAM

## RECOMP I — Aesthetic + Pull-Up Progression — 12 Weeks

**Start date:** April 2026
**Split:** Upper / Lower, 4 days/week
**Cardio:** 2x/week Zone 2 (rowing or cycling preferred — monitor right knee for running)

### Block Overview

| Block | Weeks | Focus | Rep Range | RPE Target | Sets |
|---|---|---|---|---|---|
| 1 — Volume | 1–4 | Build work capacity, calibrate weights, accumulate volume | 10–15 | 6–7 → 7–8 | 3–4 |

### Progression Rules (Per Block)

**Block 1 (Volume):** Start conservative. Week 1 is calibration — adjust any weight that's too light or heavy. Add reps before adding weight. By Week 4 you should be hitting the top of each rep range at RPE 7-8.

**Block 2 (Intensity):** Drop rep ranges, add weight. Upper compounds +5 lbs, lower compounds +10 lbs from Block 1 end weights. RPE creeps to 8-9 by Week 8.

**Block 3 (Strength):** Rep ranges drop again. Push top-end loads. RPE 8-9 across the board. Only 3 weeks — fatigue is expected.

**Block 4 (Deload):** Cut sets in half, drop weight to ~60% of Block 3. Active recovery. Log body comp, assess progress, plan next cycle.

### Pull-Up Progression Plan

| Block | Pull-Up Work |
|---|---|
| Block 1 (Wk 1–4) | Lat pulldown (building to BW equivalent), dead hangs (building to 45s+), straight arm pulldowns |
| Block 2 (Wk 5–8) | Add band-assisted pull-up negatives (3-5 sec eccentrics), continue pulldowns |
| Block 3 (Wk 9–11) | Band-assisted pull-ups for reps, test unassisted pull-up |
| Deload (Wk 12) | Light pulldowns only, test max dead hang hold |
